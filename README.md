# App String Server

## 목적

이 폴더는 **Resuscitation Skills V2 (HSTM 전용 앱)과 VCC(VivusCardioConnect)** 에서 사용되는 앱 내 모든 String을 서버로부터 수신받아 앱에서 사용할 수 있도록 관리하는 데 그 목적이 있다.

HSTM 측에서 V1 앱 내 사용되는 문구에 대한 교정 요청이 종종 발생하나, 앱 업데이트에 대한 민감도가 높기 때문에 앱 배포 없이 문구를 수정할 수 있도록 이 방편을 마련해두었다.
추가로 VCC 앱 역시 문구 정도의 수정은 앱 재배포 없이 진행할 수 있도록 폴더를 추가하였다.

---

## ⚠️ 주의사항

### 1. 파일 위치 및 구조 변경 금지
현재 별도의 URL을 설정하지 않고 `https://raw.githubusercontent.com/~` 을 그대로 사용하고 있다.
따라서 **폴더 내 파일 위치 또는 파일 형태를 절대 변경해서는 안 된다.**

### 2. 기존 Key 삭제 금지
현재 존재하는 모든 key는 앱에서 실제로 사용 중인 값이다.
**Key를 삭제할 경우 앱에서 해당 문구를 불러오지 못하는 문제가 발생한다.**

### 3. 허용되는 작업
| 작업 | 허용 여부 |
|------|-----------|
| 기존 key의 value 수정 | ✅ 허용 |
| 새로운 key-value 추가 | ✅ 허용 |
| 기존 key 삭제 | ❌ 금지 |
| 파일 위치 또는 구조 변경 | ❌ 금지 |
